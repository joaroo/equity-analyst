# Cookbook: Earnings Preview

Pre-earnings analysis for any stock ahead of a binary event.

## Invocation

```
/earnings-preview AAPL
```

## Output

- Consensus estimates (EPS, revenue, key metrics) with analyst high/low range
- Historical earnings reactions (last 4 quarters: beat/miss + stock reaction)
- Options-implied move (±%)
- Key metrics to watch for the report
- Bull / Base / Bear scenarios with specific numbers
- Positioning recommendation: Enter Before / Wait for Result / Avoid
