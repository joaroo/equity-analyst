# Cookbook: Earnings Review

Post-earnings analysis run after a company has reported results.

## Invocation

```
/earnings-review MSFT
```

## Output

- Beat/miss table (EPS, revenue, key metrics vs. consensus)
- Guidance summary (raised / maintained / lowered / withdrawn)
- Analyst reactions (upgrades, downgrades, target changes in 48h)
- Thesis impact: Strengthened / Neutral / Weakened
- Updated fundamental score (1–10)
- Action recommendation: Hold / Add / Trim / Exit + specific allocation
